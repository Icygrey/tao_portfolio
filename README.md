# Tao Portfolio

A bilingual Next.js portfolio for Tao Huang, currently focused on the homepage and resume entry points.

## Stack

- Next.js 13 App Router
- TypeScript
- CSS Modules + global CSS
- Framer Motion
- Lucide React

## Routes

- `/` English home
- `/en/resume`
- `/zh`
- `/zh/resume`

## Notes

- The current build is intentionally homepage-first.
- Footer action labels on the homepage are visual-only placeholders for future pages.
- Resume details are kept in the downloadable PDF under `public/resume`.
- Hero imagery is still placeholder-based and can be replaced later with real photos.
