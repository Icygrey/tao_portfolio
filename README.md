# Tao Portfolio

A bilingual Next.js portfolio for Tao Huang.

## Stack

- Next.js 13 App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Routes

- `/` English home
- `/en/about`
- `/en/resume`
- `/en/contact`
- `/en/brand`
- `/zh`
- `/zh/about`
- `/zh/resume`
- `/zh/contact`
- `/zh/brand`

## Notes

- The site is intentionally minimal and brand-led.
- Resume details are kept in the downloadable PDF under `public/resume`.
- Hero portraits are currently placeholders and can be replaced later with real photos.
- The anonymous board is currently a front-end preview only.
